# 👨🏻‍💻 Sobre mí

Soy **Ingeniero de Sistemas**, **Desarrollador FullStack** y **Docente Universitario**, especializado en el diseño y desarrollo de aplicaciones empresariales utilizando tecnologías modernas del ecosistema JavaScript/TypeScript.
